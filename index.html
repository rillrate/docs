Easy to start

To start the RillRate Server you can use ready-to-use Docker container:

$ docker run -d -p 1636:1636 -p 6361:6361 --restart always --name rillrate rillrate/vision:latest

When it started open the following link in your browser localhost:6361 to get access to the dashboard.
